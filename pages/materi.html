  • Percabangan If
    Pada bagian ini kita akan belajar mengenai konsep percabangan If, simak yah!
    Percabangan If adalah percabangan yang hanya akan mengerjakan instruksi apabila blok kondisi terpenuhi
    Notasi Algoritmik 
    if(blok kondisi){
        instruksi;
    }
    Flowchart
    Contoh Kasus
    Ketika kamu berjalan-jalan di sepanjang jalan kenangan, kamu tiba-tiba ingin membeli segelas kopi. Akan tetapi kamu hanya akan membeli kopi jika uang di dompet kamu lebih dari harga kopi. Kopi yang ingin beli harganya 8000, sedangkan uang di dompet kamu cuma ada 4000
    Buatlah kode program untuk contoh kasus tersebut!
    #include<iostream>
    using namespace std;

    int main(){
        int uang_di_dompet = 4000;
        int harga_kopi = 8000;

        if(uang_di_dompet > harga_kopi){
            cout << "beli kopi" << endl;
        }
        return 0;
    }
  • Percabangan If Else
    Pada bagian ini kita akan mempelajari mengenai konsep percabangan If Else
    Percabangan If Else adalah jenis percabangan yang apabila blok kondisi terpenuhi/sesuai/benar, maka instruksi akan dijalankan, akan tetapi jika blok kondisi tidak terpenuhi/salah, maka terdapat pilihan lain
    Notasi Algotimik 
    if( blok kondisi ){
       instruksi;
    }else{
       alternatif_instruksi;
    }
    Flowchart
    Contoh Kasus
    Ketika kamu menjalani ulangan, gurumu akan berkata yang nilainya lebih dari kkm maka kamu lulus tetapi apabila nilai ulangan kurang dari kkm maka kamu remedial. Nilai kamu adalah 70, sedangkan nilai kkm adalah 75. Buatlah program dalam bahasa C untuk contoh kasus ini! 
    #include<iostream>
    using namespace std;
    int main(){
        int nilai_ulangan = 70;
        int kkm = 75;
        if(nilai_ulangan >= kkm){
            cout << "lulus" << endl;
        }else{
            cout << "remedial" << endl;
        }
        return 0;
    }
  • Percabangan If Else If
    Pada bagian ini kita akan belajar mengenai konsep If Else If
    Percabangan If Else If adalah percabangan yang jika pada blok kondisi pertama belum terpenuhi maka periksa blok kondisi kedua, jika blok kondisi kedua belum terpenuhi maka periksa blok kondisi ketiga dst. Sampai kondisinya terpenuhi.
    Notasi Algoritmik 
    if( blok kondisi 1 ){
     instruksi 1;
    }else if( blok kondisi 2 ){
       instruksi 2;
    ...
    }else{
     instruksi n;
    }
    Flowchart
    Contoh Kasus
    Ketika kamu melihat nilai rapot terdapat indeks huruf. Jika kamu mendapat nilai lebih dari 85 kamu akan mendapat A, lebih dari 70 kamu akan mendapat B, lebih dari 55 kamu akan medapat C, lebih dari 40 kamu akan mendapat D, kurang dari 40 kamu mendapat E. nilaimu adalah 70. Maka, buatlah program untuk contoh kasus ini! 
    #include<iostream>
    using namespace std;
    int main(){
        int nilai = 70;
        char indeks;

        if(nilai >= 85 && nilai <=100){
            indeks = 'A';
        }else if(nilai >= 70){
            indeks = 'B';
        }else if(nilai >= 55){
            indeks = 'C';
        }else if(nilai >= 40){
            indeks = 'D';
        }else{
            indeks = 'E';
        }

        return 0;
    }
  • Percabangan Switch Case
    Pada bagian kita akan belajar mengenai konsep percabangan Switch Case
    Percabangan Switch Case adalah konsep percabangan yang memiliki banyak pilihan instruksi
    Ciri Khas Switch Case
    1. Blok kondisi hanya berisi bilangan bulat atau character
    2. Memiliki break di akhir statement
    3. Memiliki nilai default untuk mengatasi apabila tidak ada blok kondisi yang terpenuhi
    Notasi Algortimik 
    switch( menu ){
      case 1:
       instruksi 1;
        break;
      case 2:
       instruksi 2;
        break;
      ...
      case n:
       instruksi n;
        break;
       default:
        instruksi yang tidak sesuai dengan menu;
    }
    Ilustrasi Perbedaan If Else If dengan Switch Case
    Flowchart
    Contoh Kasus
    Ketika kamu datang ke rumah makan cepat saji, kamu akan mendapatkan beberapa pilihan paket menu. Pada tempat makan tersebut terdapat paket 1 yang berisi nasi, ayam goreng, kentang, minuman; paket 2 berisi nasi, 2 ayam goreng, ice cream; paket 3 berisi burger, kentang dan jus buah; paket 4 berisi nasi, ayam goreng, air mineral . Buatlah program bahasa C++ untuk contoh kasus ini! Tampilkan isi paket (kamu memilih paket 1) 
    #include<iostream>
    using namespace std;

    int main(){
        int paket = 1;

        switch(paket){
            case 1:
                cout << "Paket LAN" << endl;
                cout << "Nasi, Ayam Goreng, Kentang, Minuman" << endl;
                cout << "Harga 18k"
                break;
            case 2:
                cout << "Paket MAN" << endl;
                cout << "Nasi, 2 Ayam Goreng, Ice Cream" << endl;
                cout << "Harga 20k"
                break;
            case 3:
                cout << "Paket WAN" << endl;
                cout << "Burger, Kentang, Jus Buah" << endl;
                cout << "Harga 20k"
                break;
            case 4:
                cout << "Paket PAN" << endl;
                cout << "Nasi, Ayam Goreng, Air Mineral" << endl;
                cout << "Harga 16k"
                break;
            default:
                cout << "Paket tidak ada";
                break;
        }
        return 0;
    }
  • Percabangan Bersarang
    Kita sebentar lagi sudah mau berada di akhir materi, tidak terasa yah
    Percabangan bersarang adalah jenis percabangan yang apabila kondisi terpenuhi akan terdapat percabangan berikutnya, simpelnya, Percabangan bersarang adalah percabangan di dalam percabangan
    Contoh Kasus
    Ketika kamu ingin membeli Boba, kamu akan mendapatkan potongan seharga 5% dari total pembelian saat total pembelian kamu melewati Rp. 50.000. Diskon akan dinaikan menjadi 10%, apabila kamu sudah mendaftar sebagai member Boba. Buatlah notasi program, flowchart dan implementasi dalam bahasa C pada contoh kasus ini! (diketahui total belanja kamu sebesar 40.000 dan kamu adalah member).
